# Leadership Profile Images

## Directory Location
Place all leadership profile images in this directory:
```
frontend/public/leadership/
```

## Image File Names

Please name your images exactly as follows (case-sensitive):

1. **Ram Swaroop Yadav** (Flight Lieutenant - Retired)
   - File name: `ram-swaroop-yadav.jpg`
   - Or: `ram-swaroop-yadav.png` (if PNG format)

2. **Muneshwar Singh Yadav** (Flying Officer - Retired)
   - File name: `muneshwar-singh-yadav.jpg`
   - Or: `muneshwar-singh-yadav.png` (if PNG format)

3. **Deshraj Yadav** (Air Veteran - General Secretary)
   - File name: `deshraj-yadav.jpg`
   - Or: `deshraj-yadav.png` (if PNG format)

4. **Rajesh Kumar** (Air Veteran - Management Secretary)
   - File name: `rajesh-kumar.jpg`
   - Or: `rajesh-kumar.png` (if PNG format)

## Supported Formats
- `.jpg` or `.jpeg` (recommended)
- `.png`
- `.webp`

## Image Requirements
- **Recommended size**: 400x400 pixels or larger (square format works best)
- **Aspect ratio**: Square (1:1) is preferred for the circular profile display
- **File size**: Keep under 500KB for optimal loading

## After Uploading
Once you've placed the images in this folder with the correct names, they will automatically appear on the About page in the Leadership Team section.

## Note
If an image fails to load, a placeholder icon will be displayed instead.
